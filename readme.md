# Events Of The Day API

API for controlling daily occurrences in your work

## Applications that use it:

* <a href="https://github.com/DavidEdsonDoNascimento/events_of_the_day" target="_blank">EventsOfTheDay</a>

## Installation:

```json
npm install
```

## Before starting you need to create the database *events_of_the_day_db* and then run the migrations:

```json
npx sequelize-cli db:migrate
```

## Start:

```json
npm start
```
